Description

This is a small project for the Ludum Dare Jam 32. This time the theme was "an unconventional weapon".

This game is about a bunny who has to fight against his evil brothers using powerful fruits. You can play the web version, download a compiled build or check out the source in the section below.

If you're on Chrome, UnityWebPlayer won't work unless you do "chrome://flags/#enable-npapi" first. You can disable this after playing.

Controls

  • Move with WASD
  • Attack with the Left Mouse Buttom
  • Zoom in or out with the mouse wheel
  • Spacebar lets you jump and dive, dropping fruit from the trees.
  • Keys 1 to 3 let you swap between fruit-weapons
